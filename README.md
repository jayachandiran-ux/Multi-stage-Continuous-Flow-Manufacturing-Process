# Multi-Stage Continuous-Flow Manufacturing Process Prediction

## 📌 Overview

This repository presents a Machine Learning solution for predicting output measurements in a multi-stage continuous-flow manufacturing process using real industrial sensor data.

The manufacturing system consists of multiple machines operating in parallel and series, producing continuous process measurements. By leveraging Machine Learning, the model predicts the final output measurements, enabling better process monitoring, anomaly detection, quality control, and production optimization.

---

## 🎯 Problem Statement

Modern manufacturing industries generate large volumes of sensor data during production. Predicting the final process outputs from these measurements helps manufacturers improve operational efficiency, reduce defects, and support intelligent decision-making.

---

* Continuous-flow manufacturing process dataset
